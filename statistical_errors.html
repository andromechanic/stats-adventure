The 2x2 Gauntlet

Correct Decision

No Effect Found.

True Negative. You correctly failed to reject $H_0$.

Type I Error (\(\alpha\))

False Alarm.

Rejecting $H_0$ when it's actually true. Reporting an effect that isn't really there.

Type II Error (\(\beta\))

The Blind Spot.

Failing to reject $H_0$ when it's actually false. Missing a real effect that exists.

Statistical Power

Discovery.

True Positive. Correctly identifying a real effect. (\(1 - \beta\)).
